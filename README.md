## ts package template

Please replace 'your-package-name' in project files with your package name.

## Install

```sh
git clone https://github.com/hmmeqo/typescript-package-template.git your_package_name
cd your_package_name
pnpm install
```

## Usage

lint and fromat

```sh
# check
pnpm lint

# fix codeL
pnpm lint:fix
```

test

```sh
pnpm test
```

build

```sh
pnpm build
```

publish

```sh
pnpm publish
```
